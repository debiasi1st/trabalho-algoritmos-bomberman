# trabalho-algoritmos-bomberman

## Descrição

Projeto de implementação do jogo Bomberman como trabalho prático da disciplina de Algoritmos II.

## Objetivo

Desenvolver um jogo funcional utilizando estruturas de dados e algoritmos estudados em aula.

## Funcionalidades

- Movimentação do personagem
- Colocação de bombas
- Explosões e destruição de blocos
