# trabalho-algoritmos-bomberman

## Descrição

Projeto de implementação do jogo Bomberman como trabalho prático da disciplina de Algoritmos II.

## Objetivo

Desenvolver um jogo funcional utilizando estruturas de dados e algoritmos estudados em aula.

## Funcionalidades

- Movimentação do personagem
- Colocação de bombas
- Explosões e destruição de blocos
- Inteligência artificial para inimigos
- Sistema de pontuação

## Tecnologias

- Linguagem: C/Java/Python (conforme especificado)
- Estruturas de dados: Filas, pilhas, grafos
- Algoritmos: BFS, DFS

## Como Executar

```bash
# Clonar ou fazer download do projeto
# Compilar ou executar conforme a linguagem utilizada
make run
```
